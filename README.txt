## 使用方法

VTuber Libraryはマイナーな配信者やコンポーザーを紹介し、お勧めしあうためのプラットフォームです。以下の手順に従ってサイトを利用できます。

1. **投稿およびおすすめの配信者の追加**:
    - トップページ上の投稿ボタンから、好きなVTuberやコンポーザーを投稿してください。
    - 投稿時には、配信者の情報やプロフィール、リンクを入力することができます。リンクの入力により、画像をクリックした際に直接配信者のページに遷移できます。

2. **Elite Four ページ**:
    - Elite Four ページでは、特に有名なと思われるVTuberを「四天王」として掲載しています。

3. **ユーザー登録とログイン**:
    - 新規登録およびログインを行うことで、マイページにアクセスできます。
    - 今後のアップデートでは、投稿へのいいね機能やマイページでの参照機能などを追加予定です。


以下のリンクよりご覧ください
https://vtuber-library.onrender.com/
